Session tracking + summaries

Start a session, make changes, stop it. Trak stores a clean session record (diff stats, duration, and analysis) under .trak/.

Optional: set OPENAI_API_KEY for richer summaries and issue text.

Developer dashboard + GitHub issues

Browse sessions, drill into findings, and turn issues into GitHub tickets for follow-up.

Optional: set GITHUB_TOKEN to enable issue creation.

Demo (copy/paste)

Run Trak locally in a few minutes.

1) Install from source

git clone https://github.com/osasisorae/trak.git
cd trak
npm install
npm run build
npm link

cp .env.example .env

2) Track a session

trak start
# edit files...
trak status
trak stop

3) Open dashboards

trak dev
# opens http://localhost:3000

cd demo
node mock-org-server.js
# opens http://localhost:3001

Org reporting demo: set TRAK_ORG_ENDPOINT=http://localhost:3001, then run trak login and complete a session to POST metadata to the org dashboard.
